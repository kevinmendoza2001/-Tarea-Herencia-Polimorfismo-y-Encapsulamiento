**EJERCICIO 1
Este ejercicio consiste en crear un programa en Java que evalúe si una persona califica para un subsidio, aplicando POO mediante atributos privados,
constructores, getters y setters con validaciones, métodos para aprobar o rechazar la solicitud y sobrecarga para calcular el consumo mensual de combustible, 
usando datos ingresados por teclado en la clase Main.
**EJERCICIO 2
Este ejercicio consiste en desarrollar un sistema bancario aplicando Programación Orientada a Objetos, donde se gestionan cuentas de Ahorros, 
Corriente e Inversión mediante una clase abstracta llamada CuentaBancaria, aplicando encapsulamiento para proteger datos como saldo y titular, 
herencia para reutilizar características comunes y polimorfismo para calcular automáticamente el interés mensual de cada cuenta desde una lista
sin identificar manualmente su tipo, generando finalmente un reporte con los intereses, saldos actualizados y el total de intereses del banco.
